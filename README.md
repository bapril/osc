# osc

This is an implementation of the Open Sound Control (OSC) protocol written
in Erlang with OTP.

For more information about OSC, see the links below in the 
[Resources](#resources) section.

## Use

Compile:

    rebar3 compile

Test:

    rebar3 eunit
